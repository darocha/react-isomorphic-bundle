io.js only

---

## DEV MODE

`npm run dev`

* open `http://127.0.0.1:3000`
* open `http://127.0.0.1:3002` (BrowserSync)

## PRODUCTION MODE

`npm start`

## DEBUG MODE

`npm run debug`

* open `http://127.0.0.1:8080/debug?ws=127.0.0.1:8080&port=5858`
* open `http://127.0.0.1:3000`
