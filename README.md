# React flummox isomorphic bundle [WIP]

Working in process

```
$ npm install -g nodemon

$ npm install

```

---

## DEV MODE

`npm run dev`

* open `http://127.0.0.1:3000`
* open `http://127.0.0.1:3002` (BrowserSync)

## PRODUCTION MODE

`npm run build && npm start`

## DEBUG MODE

`npm run debug`

* open `http://127.0.0.1:8080/debug?ws=127.0.0.1:8080&port=5858`
* open `http://127.0.0.1:3000`
